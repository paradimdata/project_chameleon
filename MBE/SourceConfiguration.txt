This is the source configuration of the MBE. The port numbers here correspond to the
values in the shutter status data (e.g. GMx_Shutter_Control.Value.txt). Each shutter
port has a binary value that is added up to give the logged data value. Port 1 
corresponds to a value of 1, port 2 => 2, port 3 => 4, port 4 => 16 etc.
