===========================
Laue .hs2 X-Ray Diffraction
===========================

This module contains a function called ``hs2converter()``. This function was designed to convert Laue .hs2 X-Ray Diffraction images to .png images. The Laue images are converted from 16bpp images to 8bpp images.
