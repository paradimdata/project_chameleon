=========================================
Angle-Resolved Photoemission Spectroscopy
=========================================

This module contains mutliple functions. The primary function is  ``arpes_folder_workbook()``. This function takes an ARPES data folder and an excel workbook as input. The function parses the data from the data folder and extracts useful data and places the data in the workbook.
