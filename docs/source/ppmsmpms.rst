=============================================
Physical/Magnetic Property Management System
=============================================

This module contains a function called 'ppmsmpmsparser'. 'ppmsmpmsparser' takes a .dat file as an input. Once the user has input their file they can select what kind of run generated this file so the correct data can be extracted. All relevant data and metadata is then extracted and saved to a .txt file.
