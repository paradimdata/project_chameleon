=============================================
Physical/Magnetic Property Management System
=============================================

This module contains the function for Physical/Magnetic Property Management System (PPMS/MPMS) data. The function in this module takes a PPMS/MPMS .dat file, asks the user to pick a measurement type for the file, and extracts the relevant metadata and columns of data from the .dat file. The measurement types were created based on the measurement types of the given PPMS/MPMS machine. If your PPMS/MPMS data follows another format this function may not work for you.
